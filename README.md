# 📝 Todo App (Full-Stack)

A full-stack Todo application built using Node.js, Express, EJS, and Firebase Firestore. The app includes user authentication and allows users to manage their daily tasks efficiently.

---

## 🚀 Features

* User Authentication (Login / Signup)
* Secure user registration and login system
* Add new tasks
* View tasks in a dashboard
* Delete tasks
* Clean and simple UI using EJS templates
* Data stored in Firebase Firestore

---

## 🛠️ Technologies Used

* Node.js
* Express.js
* EJS (Embedded JavaScript Templates)
* Firebase Firestore
* HTML
* CSS

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/NagatejaThippanaboina/TodoApp
```

2. Navigate to the project folder:

```
cd TodoApp
```

3. Install dependencies:

```
npm install
```

4. Run the server:

```
node server.js
```

5. Open in browser:

```
http://localhost:3000
```
